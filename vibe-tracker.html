📋 Progress Update Prompt (copy & use with Claude Code)
以下是我的 VibeTRACKER 進度更新 prompt： ===== VIBE TRACKER UPDATE PROMPT ===== 你是我的 vibe coding 專案助手。請根據以下結構幫我更新我的進度： PROJECT_NAME: [專案名稱] TODAY_UPDATE: [今天做了什麼，1-3 句話] BLOCKERS: [遇到的問題或卡關點，沒有就填 none] NEXT_STEP: [下一步要做什麼] PROGRESS_PCT: [目前整體完成度 0-100] GOALS_COMPLETED: [今天完成的 milestone，用逗號分隔] 請回覆一個 JSON 格式，欄位如下： { "project": "專案名稱", "date": "YYYY-MM-DD", "summary": "簡短摘要", "update_text": "today_update 的內容", "progress": 數字, "goals_done": ["goal1", "goal2"], "next": "下一步" } ====================================== 使用方式： 1. 複製這個 prompt 2. 在 Claude Code / claude.ai 貼上 3. 填入你的專案資訊 4. 把回傳的 JSON 貼到 VibeTRACKER 的「＋ Log Update」
🚀 New Project Kickoff Prompt
幫我規劃一個新的 vibe coding 專案： ===== PROJECT KICKOFF ===== IDEA: [你的想法，一句話描述] GOAL: [這個專案的最終目標] TECH_STACK: [打算用什麼技術/工具] TIMELINE: [預計多久完成，e.g. 2 weeks] 請幫我： 1. 把這個想法拆成 5-8 個清楚的 milestone 2. 評估每個 milestone 的難度 (easy / medium / hard) 3. 建議完成順序 4. 給這個專案一個簡短有力的 tagline 回覆格式： { "name": "專案名稱", "tagline": "一句話說明", "goals": [ {"text": "milestone 描述", "difficulty": "easy|medium|hard"} ], "estimated_days": 數字, "tech": ["tech1", "tech2"] } ===========================
🔍 Blocker Debug Prompt
